Build a full-stack SaaS web application called **"RepurposeAI"** — an AI Content Repurposing Tool.

## 🎯 Core Idea

This app allows users to input text or a YouTube video transcript and automatically generates:

* Twitter (X) threads
* LinkedIn posts
* Instagram captions
* Blog articles
* Viral hooks

---

## 🧱 Tech Stack (STRICT REQUIREMENTS)

* Framework: Next.js 16 (App Router)
* Language: TypeScript
* Styling: Tailwind CSS
* Fonts:

  * Headings: Instrument Serif
  * Body: Inter
* Backend: Next.js API Routes (serverless functions)
* Auth: Firebase Authentication (Google login)
* Database: Firestore (store user content history)
* AI: OpenAI API (GPT-based text generation)
* Deployment: Vercel-ready (NO custom server, NO Express)

---

## ⚙️ API LOGIC

Create `/api/generate` route:

Input:
{
text: string,
tone: string
}

Use OpenAI API to generate:

### Prompts:

* Tweets:
  "Convert the following content into 5 engaging viral tweets..."

* LinkedIn:
  "Write a professional LinkedIn post with storytelling..."

* Blog:
  "Write a detailed SEO-friendly blog with headings..."

* Hooks:
  "Generate 5 viral hooks from this content..."

* Instagram:
  "Write 3 engaging Instagram captions..."

Return structured JSON:
{
tweets: [],
linkedin: "",
blog: "",
hooks: [],
captions: []
}
